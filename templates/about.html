About QuizApp

QuizApp was developed as part of the Advanced Web Technology course at Edinburgh Napier University by Lukasz Galik, a student of this institution. The task was to develop a web app in Python and Python library Flask. The requirement was to demonstrate an ability to create a dynamic website with the use of this technologies. The mark/evaluation was taking account of the ammout of functionalities and quality of the code and a solution. The following functionalities have been applied: - login/registration and connection to database - quiz game that collects user scores - results summary that uses javascript and html canvas element. Initially website has not been launched due to problems with university hosting (as part of Advanced Web Technologies) Website has been deployed thorugh Render.com now as part of Portfolio coursework for the second trimester 2024/2025.
